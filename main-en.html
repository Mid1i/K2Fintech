Automatic banking system

ABS (Automatic Banking System) - a customisable accounting and reporting system, fine-tuned to work in accordance with the international accounting standards.

The ABS is fully integrated and interfaced: General Ledger, Accounts Payable, Accounts Receivable and Fixed Assets.

  • Includes an integrated Cashbook.
  • Provides the budgeting facilities.
  • Supports Accruals/Resource based accounting practices.
  • Provides all screens, documentation and help (on screen and electronic/printed manuals) in UK/US English.

The system provides:

  • Support different second factor authentication tools

Payment module with aml

Payment Module is a module that processes all incoming and outgoing payments. It provides payment automation and integration with different payment systems.

The following operations are available under an automated AML control:

  • Internal Payments
  • SEPA external outgoing Payments (SCT)
  • SWIFT external outgoing Payments (MT103)
  • SEPA external incoming Payments (SCT)
  • SWIFT external incoming Payments (MT103)
  • Payments Cards Purchase
  • Payments Cards Cash Withdrawal
  • Payments Cards Credits

  • Postpone (conditional Payments)

  • Currency exchange (clients and internal)
